# Peta-Persebaran-Jumlah-Penduduk-Desa-Kabupaten-Purwakarta
Peta persebaran jumlah penduduk desa di Kabupaten Purwakarta Tahun 2024.

## Deskripsi
Repository ini hasil kegiatan magang yang dilaksanakan di bawah koordinasi Dinas Komunikasi dan Informatika Kabupaten Purwakarta, dengan penugasan di Dinas Pemberdayaan Masyarakat dan Desa Kabupaten Purwakarta. Peta persebaran jumlah penduduk desa disusun untuk menampilkan kondisi jumlah penduduk pada tingkat desa di Kabupaten Purwakarta. Peta ini berfungsi sebagai media visualisasi untuk menggambarkan sebaran penduduk secara spasial sebagai informasi umum wilayah.

## Informasi yang Disajikan
- Persebaran jumlah penduduk desa
- Batas administrasi wilayah desa

## Tools
- QGIS
- ArcGIS
- Microsoft Excel

## Output
- Peta Persebaran Jumlah Penduduk Desa Kabupaten Purwakarta Tahun 2024
